# CyberRecon Port Scanner

A Python-based port scanner built for cybersecurity learning.

## Features

- IPv4 Support
- IPv6 Support
- Hostname Support
- Multi-threaded Scanning
- Tkinter GUI
- Open Port Detection
- TXT and CSV Export

## Technologies

- Python
- Socket Programming
- Tkinter
- Threading
- Concurrent Futures
